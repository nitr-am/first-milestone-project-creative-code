Creative Coding... What is it?

Is a type of computer programming in which the goal is to create something expressive and not neccesarily functional.

That is the main difference it has with common programming.

It is generally used to create images, animations, visual and audiovisual art, design, architecture, installations, projections and projection mapping, advertising pieces, 3D product prototypes, fashion items and much more.

It is used by a wide range of people since software developers to artists, designers, musicians, architects and so forth.

History...

If there is one

As many of the lengages of today were created by people that today keep working, creating and updating, it is very difficult to trace in history because as the people is still writting history, old works are disappearing because a number of reasons: modification of the first websites, old software and hardware are not running any more, just to name a few.

If we really need to know the first days of creative coding we would need to hear it from the still-active creators and even doing so, we may not receive the more precise informaiton.

To sum up ang give some dates, we can say that creative code started in the 1960s decade and in the 1980s it welcomed more expert programmers who started all that we have today in the matter.

Languages:

Processing

Released in 2001, Processing is a flexible software sketchbook and open-source programming language for the visual arts. Currently Processing hasthousands of students, artists, designers, researchers, all kinds of people involved with it producing works off all types. As its original target was non-programmers, it is a very easy to learn and to use enviornment. Processing works with the Java language.

Cinder

Cinder is a C++ library for programming with cretive porpouses. It is often used to produce work in areas like graphics, audio, video, and computational geometry. Compatible with macOS, Windows, Linux, iOS, and Windows UWP and is powerful enough for professionals, and still suitable for learning and experimentation.

openFrameworks

Released in 2005, openFrameworks is a C++ open source programming language designed to assist the creative process of different kinds of people. It provides a simple and intuitive framework for experimentation. The langage has one philosiphy: it should be collaborative, usable and simple, consistent and intuitive, cross-platform, powerful, and extensible.
